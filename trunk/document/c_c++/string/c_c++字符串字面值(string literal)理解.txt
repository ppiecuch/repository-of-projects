这里所讲的字符串字面值是指形如 const char * p = "hello"的字符串。本人是C++初学者，有理解不对的地方还请大侠指教：
1 不可能有任何作用的字符串字面值不占用任何内存空间，这个估计是编译器的优化。例如以下语句，无论写多少行都不会占用内存数据段的空间。
     “hello”;
2 有可能有用的，其生存周期是整个进程。例如
      const char * p = "hello";

3 循环定义一个字符串字面值，只保留一个副本。例如下面代码，不论循环多少次p的值都不会改变。
for(int i = 0;i<100;i++)
{
    const char *p = "hello";
}
4 值相同的字符串字面值占用同一个内存空间，例如
const char *p1 = "hello";
const char *p2 = "hello";
这样p1 和p2 相同，但是这也只是编译器的优化，对于有些编译器并不优化，地址可能不同。

5 在C中字符串字面值的类型是 char *,所以其值可以改变。在C++中类型是const char *,其值不可以改变。这在编程过程中需要特别的注意，例如以下两个函数
    void fun(const char *);
    void fun(char *);
两个函数是不同的函数，稍不注意就会调用错函数。
