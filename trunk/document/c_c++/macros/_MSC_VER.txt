_MSC_VER 定义编译器的版本。下面是一些编译器版本的_MSC_VER值（参见扩展阅读中的参考文献2的链接） 　　MS VC++ 10.0 _MSC_VER = 1600 　　MS VC++ 9.0 _MSC_VER = 1500 　　MS VC++ 8.0 _MSC_VER = 1400 　　MS VC++ 7.1 _MSC_VER = 1310 　　MS VC++ 7.0 _MSC_VER = 1300 　　MS VC++ 6.0 _MSC_VER = 1200 　　MS VC++ 5.0 _MSC_VER = 1100 　　其中MS VC++ 10.0就是Visual C++ 2010，MS VC++ 9.0就是Visual C++ 2008，MS VC++ 8.0就是Visual C++ 2005。版本名称上的的对应关系参见扩展阅读中的参考文献1的链接.
